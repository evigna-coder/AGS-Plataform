# Assets para pantalla de login

Coloca aquí los archivos indicados (solo afectan la vista de inicio/login):

- **logo-anim.json** — Animación Lottie (círculo + líneas, loop infinito).
- **logo-full.png** — Imagen de respaldo si Lottie no carga (wordmark).
- **logo-icon.svg** — (Opcional) Ícono en vector.

Las rutas usadas en el código son: `/assets/logo-anim.json`, `/assets/logo-full.png`.
